SO SÁNH: MediaPipe vs OpenCV Solution
================================================================================

┌────────────────────────────────────────────────────────────────────────────┐
│                         MEDIAPIPE VERSION (ORIGINAL)                       │
│                                ❌ KHÔNG CHẠY ĐƯỢC                           │
└────────────────────────────────────────────────────────────────────────────┘

    Raspberry Pi 4 (ARM64)
           │
           │ pip install mediapipe
           │
           ▼
    ❌ ERROR: No matching distribution
    
    Nguyên nhân:
    - MediaPipe không có wheel cho ARM64
    - Build từ source rất phức tạp (Bazel, 1-3 giờ)
    - Tỷ lệ thành công thấp (~60%)


┌────────────────────────────────────────────────────────────────────────────┐
│                          OPENCV VERSION (GIẢI PHÁP)                        │
│                                ✅ CHẠY TỐT                                  │
└────────────────────────────────────────────────────────────────────────────┘

    Raspberry Pi 4 (ARM64)
           │
           │ ./install_rpi.sh
           │
           ▼
    ✅ Cài đặt thành công (5-10 phút)
    
    Lợi ích:
    - OpenCV có sẵn wheel cho ARM64
    - Dễ cài đặt, nhanh
    - Performance tốt hơn
    - Đủ chính xác cho drowsiness detection

PIPELINE COMPARISON
================================================================================

┌─────────────────────────────────────────────────────────────────────────┐
│  MediaPipe Pipeline (468 facial landmarks)                              │
└─────────────────────────────────────────────────────────────────────────┘

  Frame (640x480 BGR)
      │
      ├─── BGR → RGB conversion
      │
      ▼
  MediaPipe FaceLandmarker
      │
      ├─── Model: face_landmarker.task (~50 MB)
      ├─── Output: 468 landmarks per face
      │
      ▼
  Extract Eye Landmarks
      │
      ├─── Left eye:  16 landmarks (362, 382, 381, ...)
      ├─── Right eye: 16 landmarks (33, 7, 163, ...)
      │
      ▼
  Calculate Bounding Box
      │
      ├─── Find min/max (x,y) from landmarks
      ├─── Add padding (20%)
      │
      ▼
  Crop Eye ROI (variable size)
      │
      ├─── Left eye:  [y1:y2, x1:x2]
      ├─── Right eye: [y1:y2, x1:x2]
      │
      ▼
  RGB → Grayscale/RGB (depends on model)
      │
      ▼
  Resize to 32x32
      │
      ▼
  TFLite Eye Classifier
      │
      ├─── Model: eye_model_best.tflite (~620 KB)
      ├─── Input: [1, 32, 32, 3]
      │
      ▼
  Output: [closed_prob, open_prob]


┌─────────────────────────────────────────────────────────────────────────┐
│  OpenCV Pipeline (Face bbox + Eye detection)                            │
└─────────────────────────────────────────────────────────────────────────┘

  Frame (640x480 BGR)
      │
      ├─── Already in BGR (no conversion needed)
      │
      ▼
  OpenCV DNN Face Detector
      │
      ├─── Model: opencv_face_detector_uint8.pb (~2.7 MB)
      ├─── Config: opencv_face_detector.pbtxt
      ├─── Architecture: SSD + ResNet-10
      ├─── Input: 300x300 blob
      │
      ▼
  Face Bounding Box (x, y, w, h)
      │
      ├─── Confidence threshold: 0.5
      │
      ▼
  Extract Face ROI
      │
      ├─── Crop: frame[y:y+h, x:x+w]
      │
      ▼
  Haar Cascade Eye Detector
      │
      ├─── Model: haarcascade_eye.xml (~100 KB)
      ├─── Input: Grayscale face ROI
      ├─── Scale factor: 1.1
      ├─── Min neighbors: 5
      │
      ▼
  2 Eye Bounding Boxes
      │
      ├─── Sort by x-coordinate (left → right)
      ├─── Left eye:  (x1, y1, w1, h1)
      ├─── Right eye: (x2, y2, w2, h2)
      │
      ▼
  Crop Eye ROIs (variable size)
      │
      ├─── Left eye:  face_roi[y1:y1+h1, x1:x1+w1]
      ├─── Right eye: face_roi[y2:y2+h2, x2:x2+w2]
      │
      ▼
  BGR → RGB conversion
      │
      ▼
  Resize to 32x32
      │
      ▼
  TFLite Eye Classifier (SAME AS MEDIAPIPE)
      │
      ├─── Model: eye_model_best.tflite (~620 KB)
      ├─── Input: [1, 32, 32, 3]
      │
      ▼
  Output: [closed_prob, open_prob]
